# Integrations

Wrap all third-party services behind adapters.

## Initial Interfaces

- OpenAI.
- Twilio.
- Property data provider.
- Address validation/geocoding.
- Mapping.
- E-signature provider.
- Google advertising.
- Meta conversions.
- Email provider.
- Calendar provider.
- Object storage.
- Error monitoring.

## Controls

- Validate webhooks.
- Store raw events.
- Store external IDs.
- Use idempotency keys.
- Handle rate limits and retries.
- Track sync state.
- Provide test mode.
